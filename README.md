# burden5
>使用vue开发的个人网站项目（wordpress提供REST API）

## 项目复制
```
git clone git@github.com:WYBOOM/burden5.git
```

## 项目依赖安装
```
npm install
```

### 项目启动
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
